# FLAMES
A popular teenage match profile technique.
# Introduction
During the teenage, many of the friends use a technique to chack the match between two individuals of opposite gender.
Here, the word "FLAMES" is used which stands for "Friends", "Love", "Affection", "Marriage", "Enemy" and "Siblings".
# Procedure
The names of the two individuals are kept vertically one below the other.
The common letters are cut down in a single - out approach.
The remaining letters are counted and applied on the word "FLAMES"
# Result
The result is obtained by counting the word anti-clockwise and the particular letter is cut down where the counting stops.
The process is repeated until the last letter.
The last letter left is the final result which declares the match profile between the two individuals.
